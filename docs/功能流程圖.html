3. 警報檢查流程（每次資料更新後執行）

flowchart TD
  CALARM[" checkAlarm() "]:::start --> LOOP[" 遍歷每個選取的站點 "]:::action
  LOOP --> W{" 水位檢查 
level_in > alarmLevel?"}:::decision
  W -->|"是"| WS{" 智慧觸發 
首次 或 增幅 ≥ 0.1m?"}:::decision
  WS -->|"是"| WA[" water_level 警報 "]:::alarm
  WS -->|"否"| PUMP
  W -->|"否"| PUMP

  PUMP[" 抽水機檢查 "]:::action --> PCHK{" 狀態變化? 
0→運轉 或 運轉→0"}:::decision
  PCHK -->|"啟動"| PS[" pump_start 警報 "]:::alarm
  PCHK -->|"停止"| PE[" pump_stop 警報 "]:::alarm
  PCHK -->|"無變化"| GATE

  GATE[" 閘門檢查 "]:::action --> GCHK{" 內高外低 + 閘門全閉? "}:::decision
  GCHK -->|"是"| GA[" gate_high_inner 警報 "]:::alarm
  GCHK -->|"否"| TIDE

  TIDE[" 潮汐檢查 "]:::tide --> TICHK{" 潮汐開關啟用 
+ 外水 > 內水 + 退潮 
+ 交叉事件?"}:::decision
  TICHK -->|"是"| TO[" tide_open_gate 警報 "]:::alarm
  TICHK -->|"否"| DONE

  TO --> DONE[" 合併警報列表 "]:::action
  WA --> DONE
  PS --> DONE
  PE --> DONE
  GA --> DONE
  DONE --> AUDIO{" 新警報站點? "}:::decision
  AUDIO -->|"是"| PLAY[" playStationAlarm() 
播放警報音 + 全螢幕通知"]:::alarm
  AUDIO -->|"否 (已存在)"| KEEP
  PLAY --> KEEP[" 更新 alarmingStations 
更新 isAlarming"]:::store

4. 潮汐判斷與閘門啟閉提醒

flowchart TD
  subgraph DATA[" 資料累積 (每 30 秒) "]
    R[" 雙 API merge 後 
取 新生/建國/中山 的 level_out"]:::api
    R --> REC[" recordLevelOut() 
去重 + 保留 4 小時"]:::action
    REC --> BUF[" tideBuffer[stationno] 
滾動陣列"]:::store
  end

  subgraph CHECK[" 潮汐檢查 (每 10 分鐘) "]
    BUF --> SLICE[" 取最近 3 筆 
t-2, t-1, t"]:::action
    SLICE --> CMP{" t > t-1 且 t > t-2? "}:::decision
    CMP -->|"是"| RISE[" 漲潮 rising "]:::tide
    CMP -->|"否"| CMP2{" t < t-1 且 t < t-2? "}:::decision
    CMP2 -->|"是"| FALL[" 退潮 falling "]:::tide
    CMP2 -->|"否"| KEEP2[" 維持上次判定 "]:::action
    RISE --> SAVEDIR[" 更新 tideDirection "]:::store
    FALL --> SAVEDIR
    KEEP2 --> SAVEDIR
  end

  subgraph ALERTS[" 警報觸發 "]
    SAVEDIR --> GC{" 上次 falling 
本次 rising? "}:::decision
    GC -->|"是"| CLOSE[" tide_close_gate 
退→漲：建議關閘門"]:::alarm
    SAVEDIR --> OG{" 外水>內水 + falling 
外水剛穿過內水? "}:::decision
    OG -->|"是"| OPEN[" tide_open_gate 
交叉點：建議開閘門"]:::alarm
  end
