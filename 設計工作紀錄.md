#### TypeScript 型別問題處理

recharts 的 `Tooltip` `formatter` prop 型別為 `Formatter<ValueType, NameType>`，其中 `value` 是 `ValueType | undefined`、`name` 是 `NameType | undefined`。我們的 formatter 參數型別與之不相容。

**解決方案**: 使用 `(value: any, name: any) => [...]` 參數型別 + 整個函式後綴 `as any` 雙重繞過。

```tsx
formatter={((value: any, name: any) => {
  const numValue = Number(value);
  if (isNaN(numValue)) return ['--', name];
  return [`${numValue.toFixed(2)}m`, name === 'level_in' ? '內池' : '外池'];
}) as any}
```
